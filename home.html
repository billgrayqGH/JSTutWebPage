Introduction to Client Side Scripting JavaScript

Client side scripting refers to a class of computer programs that are executed on client side by the user's web browser engine. This type of computer programming is an important part of the Dynamic HTML, enabling web pages to have different and changing content depending on user input, environmental conditions or other variables.

JavaScript

When developing a wed page, we use HTML to formatting all document using tags which are identified and interpreted by a web browser. With CSS, we can give not only a better look to our web page but also make easy to do the maintenance but, in order to have a dynamic web page a third ingredient is needed, JavaScript. JavaScrip is a scripting language that runs on JavaScript engine of a browser and it is the most implemented and supported scripting language for the most web browsers. The implementation of JavaScript allow client side scripts to interact with the user, control the browser, communicate asynchronously, and alter the document content that is displayed.

How to code JavaScript

The are two ways to use JavaScript at client side. One way is by embedded the script within the HTML code using the tag element <script></script>. Embedded script can run in between the head or the body tags. 

				<!DOCTYPE html>
				<html>
					<head>
						<script type="text/javascript">
							alert("Your time is over!!!")
						</script>
					</head>
					<body>
					</body>
				</html>

Another way is to code it in an external file with the extension .js and then make reference on the HTML document (or documents) that is going to use the external script. 

				<!DOCTYPE html>
				<html>
					<head>
						<script type="text/javascript" src="scriptfile.js"></script>
					</head>
					<body>
					</body>
				</html>
