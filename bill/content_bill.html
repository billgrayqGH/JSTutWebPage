Java Script Data Types

What are Data Types?

Data Types are programming elements that can hold values with certain characteristics; it is used by programming languages to perform operations and output information. Data Types in Java Script is not specified when declared, the value will determine the data type.

Declaring/Initializing Data Types

Declaration

Declaration is the naming of a data type variable without having an initial value.

Initialization

Initialization is the naming of a data type variable and assigning a certain value.

Syntax

var <identifier name> = identifier value ;

Rules for Identifier Names

  • Identifier names must begin with letters
  • Identifier names can also begin with dollar sign($) and underscore( _ )
  • Identifier names can contain letters, numbers, dollar sign and underscore
  • Identifier names are case sensitive
  • Identifier names cannot be similar to reserved words or keywords
  • Identifier names cannot be operator symbols

Data Type Examples

  • Numbers − numerical values assign to a variable; it can be decimal,integers and whole numbers
    •                var quantity = 5 ;
  • Strings − set of letters, numbers, dollar signs or underscore
    •                 var bank_name  = "Scotia Bank " ;
  • Booleans − logical values of true or false
    •                  var osap_student = true ;
  • Arrays − set of elements with the same data type; they can be initialized usaing square brackets
    •   
               var t127_stud = [ "Bill","Nava Raj","Osvaldo","Kushal","Haoran" ]  ;
  • Objects − set of elements that can have different data types;Object properties are written as name:value pairs, separated by commas.
    •     
               var gb_stud = {name: "Bill", age: 21, grade: 88}  ;

Other Kinds of Data Type

  • Undefined − data type variables without assigned values
    •                   var j ;
                  var item ;
  • Empty − string variable without assigned value
    •  
                var name = " " ;
                var username  = " " ;
  • Null − variables without any values
    •   
               var field = null ;
  • Dynamic − variables that changes values
    •                   var x = 5 ;
                  var x  = 16 ;
