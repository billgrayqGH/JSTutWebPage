Java Script Loop Statements

What are Loop Statements?

Loop Statements are used in programming language to do repetitive task using lesser codes and output infinite values. There are three kinds of loop statements can be used to repeat commands on a programming language, namely: do-while, while and for-loop statements.

do-while statement

Do-while statements executes the statement inside the braces once before checking if the condition is true. When the condition is satisfied the loop will continue.

Syntax

  1. do{
  2.  
  3. <statement>
  4.  
  5. }while(condition)

Example

            <p>Click the button to loop through a block of code as long as i is less than 5.</p>
            <button onclick="myFunction()">Try it</button>
            <p id="demo"></p>

            <script>
            function myFunction()
            {
            var x="",i=0;
            do
              {
              x=x + "The number is " + i + "<br>";
              i++;
              }
            while (i<5)  
            document.getElementById("demo").innerHTML=x;
            }
            </script>

while statement

While statements checks the condition first if it is true before executing the statement inside the braces. If the condition is satisfied then the loop continues.

Syntax

  1. while(condition){
  2.  
  3. <statement>
  4.  
  5. }

Example

            <p>Click the button to loop througn a block of code as long as i is less than 10.</p>

            <button onClick="calculate()">Try it </button>

            <p id="demo"></p>
                <script>
            function calculate(){
                var text = "";
                var i = 0;
                while (i < 10) {
                            text += "<br>The number is " + i;	
                            i++;
                }
                document.getElementById("demo").innerHTML = text;
                }

            </script>

Click the button to loop througn a block of code as long as i is less than 10.

for-loop statement

For-loop statements checks the condition first if it is true before executing the statement inside the braces but there should be initial value for the counter and sometimes ending value for the counter. If the condition is satisfied then the loop continues.

Syntax

  1. for(condition){
  2.  
  3. <statement>
  4.  
  5. }

Example

            <p id="demo"></p>
    
            <script>
            cars = ["BMW", "Volvo", "Saab", "Ford"];
            text = "";
            var i;
            for (i = 0; i < cars.length; i ++) {
                    text += cars[i] + "<br>";	
            }

            document.getElementById("demo").innerHTML = text;
            </script>
