JavaScript Conditional Statements

In programming language conditional statements preform different action based upon conditions. Following are Conditional Statement in JavaScript language

  • if statement
  • if else statement
  • else statement
  • Switch statement

if, if else, and else statement

These statements give decision according to true or false of condition.” if” and “if else” statements run only the logical condition is true otherwise else condition run.

Syntax

  1. if (condition) {
  2. Run this code ;
  3. }
  4. else if { if the first statement is false then run this code
  5. }
  6. else { if all above statement are false then run this code
  7. }

Switch statement

This is like as if statement, but it can check multiple value at a same time

Syntax

  1. Switch (expression) {
  2. case 1 :Run this code ;
  3. break;
  4. case 2 :Run this code ;
  5. break;
  6. case 3 :Run this code ;
  7. break;
  8. default: Run this code;
  9. Break;
  10. }
